# 🌐 Доступ к фронтенду

## ✅ Правильный URL

```
http://localhost:8080/
```

## ❌ НЕПРАВИЛЬНЫЕ URL

- `http://localhost:8080/api/posts` - это API, возвращает JSON
- Это НЕ фронтенд!

## 🎯 Что вы видите:

| URL | Тип | Результат |
|-----|-----|-----------|
| `http://localhost:8080/` | Frontend | 🎨 Красивый интерфейс с постами |
| `http://localhost:8080/posts` | Frontend | 🎨 Та же страница |
| `http://localhost:8080/api/posts` | API | 📊 JSON данные (используется фронтендом) |

## 🚀 Как получить доступ

1. Убедитесь что приложение запущено:
   ```bash
   mvn spring-boot:run
   ```

2. Откройте в браузере:
   ```
   http://localhost:8080/
   ```

3. Вы должны увидеть:
   - 📝 Заголовок "Blog Posts"
   - 🔍 Поле поиска
   - ✏️ Кнопка "Create New Post"
   - 🚪 Кнопка "Logout"
   - 📋 Список всех постов в виде карточек

## 🔧 Структура URL

```
Root URL: http://localhost:8080
├── /                    → HTML страница (фронтенд)
├── /posts              → HTML страница (фронтенд)
├── /api/posts          → JSON API endpoint
├── /api/posts/{id}     → JSON API для конкретного поста
└── /css/, /js/         → Статические файлы
```

## 📝 Запомните

- `/api/*` = **API endpoints** (возвращают JSON)
- `/` = **Frontend** (возвращает HTML)

Если вы видите JSON в браузере при посещении `/api/posts` - это нормально! 
Это означает что ваш API работает. 

Фронтенд автоматически будет использовать этот API для загрузки данных.
